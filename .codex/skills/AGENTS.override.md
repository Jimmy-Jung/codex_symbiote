# .codex/skills Scope Override

이 파일은 `.codex/skills/` 하위에서 작업할 때만 적용됩니다.

## 스킬 작성 원칙

- 각 스킬은 하나의 명확한 문제를 해결해야 합니다.
- `SKILL.md`의 설명은 "언제 사용하는지"를 한 문장으로 명시합니다.
- 스킬 내부 스크립트/템플릿이 있으면 우선 재사용하고, 중복 구현을 피합니다.
- 스킬 문서는 실행 가능한 절차 중심으로 유지합니다.

## 품질 체크

- 경로 참조는 실제 파일 기준으로 검증합니다.
- 예시 커맨드는 현재 Codex CLI 표기와 일치해야 합니다.
- 오래된 툴 키(`mcpServers`, `on-failure`)가 남아 있지 않은지 확인합니다.
